# AI Tooling

This project was planned and implemented with assistance from Codex. Codex helped interpret the assignment rubric, choose a low-cost RAG architecture, scaffold the Flask app, create synthetic Code 5 Developers operating documents, write tests, add the document library, and prepare documentation.

The most useful AI assistance was rapid generation of boilerplate, test cases, PDF/manual drafts, and UI structure. Human review is still needed for final evaluation results, Render secret configuration, the demo recording, and verifying that all submitted links are accessible to the grader.
