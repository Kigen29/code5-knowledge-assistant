# Demo Script

1. Open the deployed app or local `http://127.0.0.1:5000`.
2. Show the Code 5 Developers branding and indexed document count.
3. Open `/library` and read one Markdown policy and one PDF manual.
4. Ask: "What is Code 5 Developers' standard software delivery workflow?"
5. Point out the separate citations and retrieved snippets.
6. Ask: "How quickly are critical production incidents acknowledged?"
7. Ask: "What should be included in a maintenance handover?"
8. Ask an unrelated question and show the refusal guardrail.
9. Show `design-and-evaluation.md` and the latest evaluation metrics.
10. Show the GitHub Actions CI run and Render deployment.
